Programming Languages Overview

HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

Basic structure:



  
    Page Title
  
  
    HTML (HyperText Markup Language):
 HTML is the standard markup language used to create and design the structure of web pages. It works as the backbone of every website and determines how content is displayed in a browser.
 Key Features:
- Elements & Tags: HTML uses elements like html, head>, and body to define the structure. Tags like h1, p, and a create headings, paragraphs, and links.
- Attributes: Tags can have attributes (e.g., href, src, class) to define additional properties.
- Media Support: Incorporates images (img), videos video, and audio audio seamlessly.
- Linking: Allows linking to other web pages or resources using a tags.
- Forms: Collect user input through form elements with various input types.

HTML does not have input or syntax like programming languages; it defines the structure of web content.

CSS

CSS (Cascading Style Sheets) is used to style HTML elements on a web page.

Basic syntax:

selector {
  property: value;
}

Example:

body {
  background-color: lightblue;
}

Java

Java is a popular object-oriented programming language used for many types of applications.

Basic program structure:

public class Main {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}

Input example: Using Scanner class

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
System.out.print("Enter your name: ");
String name = scanner.nextLine();
System.out.println("Hello, " + name);

JavaScript

JavaScript is a versatile scripting language primarily used for web development.

Basic program example:

console.log("Hello, World!");

Input example: Using prompt()

let name = prompt("Enter your name:");
console.log("Hello, " + name);

C

C is a powerful procedural programming language widely used for system programming.

Basic program structure:

#include 

int main() {
    printf("Hello, World!\n");
    return 0;
}

Input example:

int number;
printf("Enter a number: ");
scanf("%d", &number);
printf("You entered: %d\n", number);

Python

Python is a high-level, interpreted programming language known for its readability.

Basic program example:

print("Hello, World!")

Input example:

name = input("Enter your name: ")
print("Hello, " + name)
